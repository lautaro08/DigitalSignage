Cartelería Digital

Abrir Ventana Publicitaria

Permite abrir la cartelería disponible y programada, en la cual se puede observar una o varias campañas publicadas.

Administrador de Campañas

Permite gestionar las campañas desde su concepción. Se puede filtrar (todas las campañas, campañas activas hoy, campañas activas en la fecha, campañas con el ID, campañas con el nombre) el listado de campañas para que se despliegue una tabla (id, nombre, descripcion), en la que se puede observar las campañas activas, puediendo agregar, modificar o eliminar las mismas.

Administrador de Banners

Permite gestionar los banner desde su concepción. Se puede filtrar (todas los banners, banners activos hoy, banners activos en la fecha, banners con el ID, banners con el nombre) el listado de banners para que se despliegue una tabla (id, nombre, descripcion), en la que se puede observar los banners activos, puediendo agregar, modificar o eliminar los mismos.

Administrador de Fuentes RSS

Permite gestionar fuentes RSS agregando, modificando y eliminando las mismas. Las fuentes gestionadas se pueden observar en un formato de tabla donde se encuentran ordenadas segun su ID y descripcion, puediendo filtrar (todas las fuentes RSS, fuentes RSS con el ID) la búsquedas de las mismas.
